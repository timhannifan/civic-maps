About

The ability to create rich, useful maps is a critical skill for web developers, data scientists, and civic hackers. This site contains examples created with Mapbox GL JS.

Usage

To run this version locally or create your own map portfolio, clone this repo and run the execution script below. More instructions on how to set up a JKAN/Jekyll instance can be found here.

bundle exec jekyll serve
